About

I'm Eric Price, a freelance web designer and developer in the Raleigh area of North Carolina, specializing in responsive web design. I have been coding both personally and professionally for years and the last few years have been focused primarily on web development.

I have a clean design style where content works with the overall design, not against it. That style also applies to my coding, with clean code that is optimized for performance. My design process from the beginning accounts for all screen sizes, so it's not necessary to remove or adjust content to fit a mobile screen.

My current skillset is HTML, CSS, SASS, jQuery, UX design and constantly looking to improve and build upon it every day. I'm always looking to start a project, so drop me a line if you want to start yours.
